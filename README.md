# Scrum - Projektmanagement  
Im Rahmen unseres Gruppenprojekts im Modul Data Mining wird die Entwicklung und Umsetzung unseres Konzepts mithilfe der Scrum-Methode organisiert.  

Zur Koordination der Zusammenarbeit haben wir ein Google-Dokument erstellt, das einen strukturierten Überblick über unseren aktuellen Projektfortschritt sowie die anstehenden Aufgaben bietet. Außerdem haben wir uns am Anfang zusammengesetzt, sodass wir unsere Rollen feststellen können.
